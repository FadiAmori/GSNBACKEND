## Environment Variables Reference

**Copy this, fill in blanks, paste into platform:**

```
ASPNETCORE_ENVIRONMENT=Production

ConnectionStrings__DefaultConnection=postgresql://[YOUR_DB_USER]:[PASSWORD]@[HOST]:5432/[DATABASE]

JwtSettings__SecretKey=[YOUR_GENERATED_SECRET_32_CHARS_MIN]

JwtSettings__Issuer=BudgetBackend

JwtSettings__Audience=BudgetApp

Cors__AllowedOrigins__0=https://your-frontend-url.vercel.app

Logging__LogLevel__Default=Information

Logging__LogLevel__Microsoft=Warning
```

## If Something Goes Wrong

```
CORS Error?
?? Check Cors__AllowedOrigins__0 = exact frontend URL
?? No trailing slash
?? Include protocol (https://)
?? Example: https://example.vercel.app

JWT Error?
?? Check JwtSettings__SecretKey is same as used at login
?? Check token not expired (8 hours)
?? Check token format: Bearer TOKEN

Database Error?
?? Check connection string format
?? Check PostgreSQL is running
?? Test connection string locally first
?? Check firewall settings

Still stuck?
?? See DEPLOYMENT_QUICK_REFERENCE.md Troubleshooting
```
